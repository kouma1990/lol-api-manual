GET /api/lol/{region}/v1.2/championbbbbbb

RESPONSE CLASSES

Return value: ChampionListDto

ChampionListDto - This object contains a collection of champion information.
NAME DATA TYPE DESCRIPTION
champions List[ChampionDto] The collection of champion information.


ChampionDto - This object contains champion information.
NAME DATA TYPE DESCRIPTION
rankedPlayEnabled boolean Ranked play enabled flag.
botMmEnabled boolean Bot Match Made enabled flag (for Co-op vs. AI games).
botEnabled boolean Bot enabled flag (for custom games).
active boolean Indicates if the champion is active.
freeToPlay boolean Indicates if the champion is free to play. Free to play champions are rotated periodically.
id boolean Champion ID. For static information correlating to champion IDs, please refer to the LoL Static Data API.


RESPONSE ERRORS
HTTP STATUS CODE REASON
400 Bad request
500 Internal server error
429 Rate limit exceeded
503 Service unavailable
401 Unauthorized


QUERY PARAMETERS
NAME VALUE DATA TYPE DESCRIPTION
freeToPlay (optional) boolean Optional filter param to retrieve only free to play champions.
